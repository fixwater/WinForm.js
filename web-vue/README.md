# web-vue

构建中...

## 安装依赖

**npm**

```
npm install
```

**yarn**

```
yarn install
```

## 命令

**npm**
#### 启动项目

```
npm run serve
```

#### 打包项目

```
npm run build
```

**yarn**
#### 启动项目

```
yarn serve
```

#### 打包项目

```
yarn build
```

详细请见 `package.json`
